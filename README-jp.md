## はじめに
### 必要なもの
文書を読むため, ゲームのリプレイを見るため,
そしてゲームの競技場の構成を編集するのにウェブブラウザが必要です.

準備中のゲーム管理システムでは C++ 開発環境
(C++14 以上のコンパイラと標準ライブラリ) を利用する予定です.
